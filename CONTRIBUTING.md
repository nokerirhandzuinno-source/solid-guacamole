## 🧭 Contribution Guidelines

### 1. 🔍 Explore the Ritual Scrolls
- Read the `README.md` and Wiki pages to understand the mythic system.
- Review open Issues and Discussions to find active glyphs needing attention.

### 2. 🛠️ Fork & Clone
- Fork this repository to your own GitHub account.
- Clone it locally and create a **ritual branch** (e.g., `ritual-badge-initiate`, `fix-glyph-alignment`).

### 3. 🧪 Ritual Hygiene
- Follow clean commit practices:  
  `git commit -m "🧬 Add badge system for Initiates"`  
- Use descriptive PR titles and include a summary of your glyph.

### 4. 🧵 Pull Request Protocol
- Open a pull request to the `main` or designated ritual branch.
- Include:
  - What you changed
  - Why it matters
  - Any symbolic or mythic context
- Be respectful, clear, and open to feedback.

### 5. 🧙‍♂️ Review & Merge
- All merges are **sovereign** and **ritualized**.
- Maintainers will review your glyph and guide it to ascension.
