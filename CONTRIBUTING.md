## 🧭 Contribution Rituals

### 🔍 1. Read the Scrolls
- Begin with the `README.md`, Wiki pages, and Discussions.
- Understand the mythic structure, emblem, and founder’s declaration.
- Review open Issues labeled `🧬 glyph`, `🛡️ defense`, or `🎨 ritual`.

### 🛠️ 2. Fork & Clone
- Fork this repository to your GitHub account.
- Clone it locally and create a **ritual branch**:
  - `ritual-badge-initiate`
  - `fix-glyph-alignment`
  - `vundare-scroll-update`

### 🧪 3. Commit with Symbolism
- Use clear, expressive commit messages:
  - `git commit -m "🧬 Add badge system for Initiates"`
  - `git commit -m "🛡️ Harden relay node with IP allow list"`
  - `git commit -m "🎨 Embed MVN emblem into landing page"`

### 🧵 4. Pull Request Protocol
- Open a pull request to `main` or designated ritual branch.
- Include:
  - What you changed
  - Why it matters
  - Any symbolic or mythic context
- Be respectful, clear, and open to feedback.  
  This is a **collaborative ritual**, not a competition.

### 🧙‍♂️ 5. Review & Merge
- All merges are **sovereign** and **ritualized**.
- Maintainers will review your glyph and guide it to ascension.
- You may be invited to join the **Vundaré Circle**—a private channel for trusted glyphsmiths.
